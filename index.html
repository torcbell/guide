4. Διαλύστε τον χρόνο σας και αφήστε το να είναι πραγματικός χρόνος. Ο πραγματικός χρόνος βρίσκεται όταν τα πράγματα συμβαίνουν σε πραγματικούς χώρους. Δεν έχει καμία σχέση με την εβδομάδα εργασίας, τον ενοποιημένο χρόνο της φορητής συσκευής. Έχει ακόμη λιγότερη σχέση με τις ώρες γραφείου, τις χαρούμενες ώρες, την ώρα θεραπείας. Ό,τι κι αν συμβεί, θα συμβεί στον φυσικό του χρόνο. Υποθέστε ότι εξετάζετε πόσο χρόνο χρειάζεται για να φέρει ένα παιδί να τελειοποιήσει ή να γράψει ένα μυθιστόρημα. Λοιπόν, αν ήσασταν σε κάποιο είδος «πεδίου», σε αντίθεση με το Plan-it Earth, ένα από αυτά θα έπρεπε να περιμένει μέχρι το άλλο έγινε. Βιολογικά ρολόγια, μακροχρόνιες σχέσεις, που θα καθορίσουν ποια θα έρθει πρώτη. Φυσικά, μπορείς πάντα να παντρευτείς πλούσιους, αλλά (εκτός αν κάποιος χρειάζεται υπηκοότητα) γιατί να ασχοληθείς; Είναι σημαντικό να μην ξεμένεις από βενζίνη αλλά να μην περιοδεύεις σε βενζινάδικα. 5. Λοιπόν ας υποθέσουμε ότι αυτές οι καταστάσεις έκαναν τον κόσμο σας; Το συλλογικό πράγμα θα ήταν εύκολο να γίνει αν όλοι αξιοποιήσουν τον πλούτο των κοινοτήτων, επειδή τα άτομα είναι φτωχά συγκριτικά. Ζητήστε πιρούνι ιδιοκτήτη σας πάνω από έναν κήπο της κοινότητας. Χρησιμοποιήστε τις εποχές των πραγμάτων για να χαρτογραφήσετε μια πορεία για το κοινωνικό. Η θεωρία των εναλλασσόμενων αντιθέσεων, όπως η νύχτα—ημέρα, νύχτα—δράση, ησυχία—δράση θα μπορούσε να σας αναγκάσει να οργανώσετε τη συγκυρία των γεγονότων στενά μαζί με τη σειρά των πρεμιέρων, εκλογές, εβδομάδα μόδας. Ένα σύμπαν. Αλλά αυτό μπορεί να είναι άβολο, έτσι ώστε να τους χωρίζουν με οποιονδήποτε τρόπο είναι καλύτερο για τους συμμετέχοντες, ένα οικονομικό τρίμηνο χώρια αν είναι απαραίτητο. Μιας και είσαι στον κόσμο τώρα και όχι στο μεταπτυχιακό, δεν μπορείς να έχεις οικονομικό κόστος ένα άθλιο άθλημα. Αν η εικόνα σας καλεί τον πρόεδρο και τους αντιπροέδρους του Τσέις-Μανχάταν να κάθονται στο μεγαλύτερο θησαυροφυλάκιο τους και να πετάνε χρυσά νομίσματα γύρω σαν μωρά και δεν μπορείτε να τους πείσετε να το κάνουν, τότε ξεχάστε το και πηγαίνετε σε κάτι άλλο. Αν χρειαστεί να κόψετε πολλά ξύλα με αλυσοπρίονα κλαψουρίζοντας και δέντρα ραγίζουν, βρείτε έναν τύπο που χρειάζεται κάποια ξύλα αραιωμένα ούτως ή άλλως. Αν θέλετε μια μπουλντόζα να μασήσει κάποιο έδαφος, να ξεκινήσει μια περιοχή τεχνών. Θα σωθείς από ένα χρέος μιας ζωής και μπορεί να μάθεις κάτι για την ισοπέδωση του εδάφους. Αν θέλετε να δουλέψετε με παιδιά, ανακαλύψτε τι πραγματικά μπορούν να κάνουν και τι θέλουν αντί να τους δίνετε κάτι φανταχτερό που θα θέλατε να κάνετε αλλά δε θα το κάνετε. Ας φτιάξουν κάτι μέσα από σωρούς σκουπιδιών, ας βάψουν μερικά παλιά αυτοκίνητα σε μια μάντρα, και ας σκάψουν μια τεράστια τρύπα στην παραλία. Αναθέτουν την οικοδόμηση του κόσμου σε μια σοφότερη γενιά. Αν θέλετε όλοι οι συμμετέχοντες να ξεκινήσουν γυμνοί, κολυμπώντας, κάνοντας έρωτα ή οτιδήποτε άλλο, υπάρχουν στιγμές και πολλά μέρη όπου δεν θα ξεσηκώσει σκόνη. Από την άλλη, αν σου αρέσει να σε πιάνουν οι μπάτσοι, μπορεί να σκεφτείς να βάλεις φυλακή στην κοσμοθεωρία σου. 6. Κόσμος με τη δύναμη γύρω σας, όχι ενάντια σε αυτό. Κάνει τα πράγματα πολύ πιο εύκολα και σας ενδιαφέρει να κάνετε τα πράγματα πιο εύκολα. Όταν χρειάζεστε επίσημη έγκριση, βγείτε έξω. Μπορείς να χρησιμοποιήσεις την βοήθεια της αστυνομίας, του δημάρχου, του πρύτανη του κολεγίου από το οποίο παράτησες, του εμπορικού επιμελητηρίου, της διοίκησης επιχειρήσεων, των κεφαλαιούχων και όλων των φίλων σου. Γίνε το δικό σου πρόσωπο δημοσίων σχέσεων. πείστε τους όλους ότι αυτό που κάνετε αξίζει τον κόπο γιατί είναι ευχάριστο να παίζετε, όπως είναι και ευχάριστο για αυτούς να πάνε για ψάρεμα. Φυσικά, δεν είναι στιγμιότυπο, αλλά σε όλους αρέσει να χειραγωγούνται. Τσαντίζονται όταν το κάνεις άσχημα. Όταν έχεις κάτι να σε βοηθήσει,μην κάνεις πρόβα τι να πεις όταν οι άνθρωποι σε ρωτάνε τι κάνεις. Αυτό θα το καταστήσει αφύσικο επειδή θα χτίσει στην ιδέα της νομιμότητας, κάποια a priori αντίληψη του κόσμου. Δεν υπάρχει τίποτα για να βελτιώσει την πορεία σας, δεν χρειάζεται να είναι ένας επαγγελματίας εκτελεστής. Είναι καλύτερο όταν είναι άτεχνο, για το καλύτερο ή το χειρότερο. Αν δεν λειτουργεί, μπορείτε να αλλάξετε τις απαντήσεις που παίρνετε ανακατεύοντας τις γλώσσες που χρησιμοποιείτε για να υποβάλετε την ερώτηση. "Τι κάνεις;" — «...εννοείτε εδώ;» είναι περιττό να προβάρεις καταστάσεις όπως το να κοιτάζεις κάτω από το βαρέλι των τριάντα σου, να γκρεμίζεις μια δημόσια απειλή, να πετάς αντίγραφα σε ένα λιβάδι και να βλέπεις τη βροχή να ξεπλένει το μελάνι, οδηγώντας ένα σωρό αυτοκίνητα σε διαφορετικές κατευθύνσεις μέχρι να τελειώσουν από βενζίνη. Αυτές δεν είναι τελειοποιημένες ενέργειες. 7. Απλά το χτίζω μια φορά. Αλλιώς θα έχεις πολύ περισσότερα σκουπίδια, και όχι μόνο το είδος των B.S. που υπονοείται εδώ. Αν δεν βελτιώσεις τους Κοινούς Κόσμους, θα είναι δύσκολο να χτίσεις κάτι. Φανταστείτε να προσπαθείτε να πάρετε αντίγραφα των παλιών σας αντιγράφων, για να δείτε τη βροχή να ξεπλύνει αυτούς τους τρυφερούς βαθμούς, ενώ στέκεστε εκεί μόνοι σας αναρωτιέστε πού είναι όλοι οι συνομηλίκοι σας. Το πανεπιστήμιο δεν είναι ένας κόσμος. Γιατί ενοχλείς; Παράτα την ιδέα να γίνεις επαγγελματίας, να γίνεις viral, να γίνεις δημόσια. Όλος ο κόσμος είναι μια σκηνή, αλλά όχι ο κάθε κόσμος. Αφήστε τις εκπομπές στους σημαντικούς ανθρώπους και στον Jeff Bezos επειδή είναι πραγματικά το κοινό που κινεί τα νήματα. Το Worldbuilding είναι ένα παιχνίδι με υψηλό, μια τελετουργία που καμία εκκλησία δεν θα ήθελε γιατί δεν υπάρχει θρησκεία προς πώληση. Το Worldbuilding είναι για εκείνους που συμβαίνουν σε αυτόν τον κόσμο, για εκείνους που δεν θέλουν να πάνε στο σχολείο και να έχουν ένα ορισμένο είδος ασφάλειας, ήδη ξεπερασμένη. Αν θέλεις να κάνεις το παν, δεν μπορείς να είσαι έξω και να κρυφοκοιτάζεις μέσα. Πρέπει να εμπλέκεσαι σωματικά. Χωρίς αφεντικό, σύμβουλο ή υποτροφία, μπορείς να είσαι σε κίνηση, χρησιμοποιώντας κάθε είδους περιβάλλον, να ανακατεύεις φυτά και ζώα, να μην ανησυχείς για το τι θα σκεφτούν οι καθηγητές σου στο γυμνάσιο, και μπορείς να διαδώσεις τη δράση σου σε όλο τον κόσμο όποτε θέλεις. Ο επαγγελματισμός είναι σαν τη σχολική εκπαίδευση και τα ναρκωτικά: είναι τροφή για τους ανθρώπους που πρέπει να κάτσουν στον πισινό τους για περισσότερο και περισσότερο χρόνο για να πάρουν το νόημα, και το θέμα είναι ότι υπάρχουν πολλά άλλα πράγματα να κάνουμε, τα οποία όλοι οι έξυπνοι άνθρωποι προτιμούν να σκέφτονται μόνο. Αλλά σοβαρές οικοδόμοι κόσμο έχουν ένα σχέδιο και να προχωρήσει και να το πραγματοποιήσει. Χρησιμοποιώντας μια παλιά έκφραση, δεν φτιάχνουν απλώς τον κόσμο, τον κάνουν καλύτερο απ' ότι τον βρήκαν.

2.建立你在现实世界中看到的世界；世界建设真正的地方和人而不是学校。如果你继续上学，最终你会得到一些无聊的机会，因为那是一个庞氏骗局。利用现有的“程序”：学生运动，防暴警察，催泪弹，要求，灌水 — 这些是生态系统。还记得他们在祖科蒂公园像分发糖果一样分发比特币的时候吗？不要犹豫，要成为追逐风暴的人，从烟山到立石山寻找灾祸。你用iphone能做很多事。别费心去研究机构资源：试试握手协议。一个机构只是一代代的内容。要记住，唯一比钱更容易的是人。你只要互动就能赚很多钱。 3.打破你的朋友圈。演员们依赖的是一个普通的演员阵容。你可以通过逐渐拓宽朋友之间的社会经济视野来进行试验；首先是相似利益层面，然后是年龄、邻域、时区、城市、大陆、世界、星系之间。你不必同时出现在任何地方。你甚至不必身处一个著名的美国城市，更别提这座城市了。你所处的地方和其他参与者所在的地方一样好。学会认清这一点。世界建设不是为了建设新的世界。就是让已经存在的东西变得宜居。所以要避免讲台、贸易展、名片、简历、流行饮食、生活教练、亲爱主题者、社交媒体正义运动等等。如果有人要你钱，就把钱给他们。如果你住在这类城市，这种生活来源会更多。这类地方太贵了，不宜过日子。当你想要一个新的项目时，最好是停留，直到有人想出一个有趣的问题，然后是一个更有趣的问题。一群艺术家被赶出了一个合作社，然后，一个自然形成的文化飞地 — 那里居住着第一代美国人 — 开始进入人们的意识。这两个团体意识到他们有一些共同点，开始参与地区政治。当地的一家印刷厂对这个原因表示同情。有人捐赠了很多电脑。 4.打发时间，让时间成为现实。实时性是在真实的地方发生时发现的。它与工作周、移动设备的统一时间无关。这跟办公时间，快乐时间，治疗时间关系更小。不管发生什么，都应该在自然时代发生。假设你想想把一个孩子带去上学或写一本小说需要多长时间。如果你在某种“领域”，而不是地球计划，其中一个必须等到另一个完成。生物钟，长期关系，决定谁先来。当然，你总是可以娶富婆，但是（除非有人需要公民身份）为什么还要费心呢？重要的是不要耗尽燃气，但是你不能到加油站去旅行。 5.那么，假设这些情况组成了你的世界？如果每个人都能利用社区的财富，集体行动将是件容易的事，因为个人相对来说已经破产了。要求房东在社区花园叉子。利用事物的季节来制定社会的发展路线。交替对比的理论，比如说夜晚，夜晚，行动，安静，行动，可以让你按照首相的顺序，选举，时装周来严密地组织事件的集中。一个宇宙。但这可能有些不方便，所以要用对参与者来说最好的方式把它们捆起来，必要时可以分开一个财务季度。既然你现在已经身居世界，没有念过研究生，你就无法承担成为一项糟糕的运动了。如果你的照片呼唤大通曼哈顿的总裁和副总裁，坐在他们最大的保险库里，像婴儿一样扔金币，然后你不能让他们这么做，那就忘了它然后去做别的事情。如果你需要砍伐大量木材，用锯链锯来锯齿，树木就会开裂，那么还是要找到一个需要把树苗砍掉的人。如果你想用推土机铲平地面，那就开个艺术区。你会为自己省下一辈子的债务，或许还能学到些地平线上的东西。如果你想和孩子一起工作，那就去了解他们真正能做和喜欢的事情，而不是给他们一些你想做、但不想做的花哨的东西。让他们用一堆堆的垃圾建造些东西，把一些旧车粉刷在垃圾堆里，在海滩上挖一个大洞。将世界建设外包给更加明智的一代。如果你想让所有的参与者都开始裸体、游泳、做爱或者别的什么，那么有的时候，也有很多地方它不会引发任何尘埃。另一方面，如果你喜欢被警察逮捕，你可能会考虑把监狱放进你的世界观。 6.世界有你周围的力量，而不是反对它。它让事情变得容易做起来你很想把事情做完当你需要官方批准时，就去申请吧。你可以动用警察的力量，市长，从学校辍学的院长，商会，公司主管，风险投资家和你的朋友。做你自己的公关人物；让他们相信你在做的事是值得的，因为玩的很开心，就像他们去钓鱼一样。当然，这并不是一瞬间，但所有人都喜欢被操纵。你做得不好，他们就会生气。当你有东西要对你做时，不要排练当别人问你要做什么时该说些什么。这将使它变得不自然，因为它将建立在合法性的理念上，也就是对世界的先验概念。你的道路没有什么可改进的，你不需要成为一个专业的表演者。不管好坏，当它没有技巧时，它是最佳的选择。如果它不起作用，你可以改变你得到的答案，你可以改变你问问题的语言。“你是做什么的？”
